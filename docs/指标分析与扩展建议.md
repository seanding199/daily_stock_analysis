## 🚀 建议新增技术指标

### 🔴 高优先级（强烈推荐）

#### 1. **布林带（BOLL）- 波动率指标**
**用途**：判断价格波动区间和超买超卖
```python
# 计算逻辑
中轨 = MA20
上轨 = MA20 + 2 * 标准差
下轨 = MA20 - 2 * 标准差
```
**交易信号**：
- 价格触及下轨 → 超卖，可能反弹
- 价格触及上轨 → 超买，可能回调
- 布林带收窄 → 变盘前兆
- 布林带张口 → 趋势延续

**实现难度**：⭐⭐ 简单
**实用价值**：⭐⭐⭐⭐⭐

---

#### 2. **KDJ 指标（随机指标）**
**用途**：短线买卖点判断，比 RSI 更敏感
```python
RSV = (收盘价 - 最低价) / (最高价 - 最低价) * 100
K = RSV的3日平滑移动平均
D = K的3日平滑移动平均
J = 3K - 2D
```
**交易信号**：
- K线上穿D线 → 金叉（买入）
- K线下穿D线 → 死叉（卖出）
- J值 > 100 → 超买
- J值 < 0 → 超卖

**实现难度**：⭐⭐ 简单
**实用价值**：⭐⭐⭐⭐⭐

---

#### 3. **ATR（平均真实波幅）**
**用途**：衡量价格波动性，设置止损点位
```python
TR = max(
    最高价 - 最低价,
    abs(最高价 - 昨收),
    abs(最低价 - 昨收)
)
ATR = TR的14日移动平均
```
**应用**：
- 止损位 = 买入价 - 2*ATR
- 止盈位 = 买入价 + 3*ATR
- 波动率过大 → 风险提示

**实现难度**：⭐⭐ 简单
**实用价值**：⭐⭐⭐⭐⭐

---

#### 4. **成交量加权均价（VWAP）**
**用途**：机构成本参考线
```python
VWAP = Σ(价格 × 成交量) / Σ成交量
```
**交易信号**：
- 价格 > VWAP → 强势，机构盈利
- 价格 < VWAP → 弱势，机构亏损
- 回踩VWAP不破 → 支撑有效

**实现难度**：⭐⭐ 简单
**实用价值**：⭐⭐⭐⭐

---

### 🟡 中优先级（推荐）

#### 5. **CCI（顺势指标）**
**用途**：超买超卖 + 趋势判断
- CCI > +100 → 超买
- CCI < -100 → 超卖
- 零线上下穿 → 趋势转换

**实现难度**：⭐⭐⭐ 中等
**实用价值**：⭐⭐⭐⭐

---

#### 6. **OBV（能量潮）**
**用途**：判断资金流向
```python
涨 → OBV += 成交量
跌 → OBV -= 成交量
```
**交易信号**：
- 价格创新高，OBV未创新高 → 顶背离（看空）
- 价格创新低，OBV未创新低 → 底背离（看多）

**实现难度**：⭐⭐ 简单
**实用价值**：⭐⭐⭐⭐

---

#### 7. **威廉指标（WR）**
**用途**：超买超卖判断
```python
WR = (最高价 - 收盘价) / (最高价 - 最低价) * 100
```
- WR < 20 → 超买
- WR > 80 → 超卖

**实现难度**：⭐⭐ 简单
**实用价值**：⭐⭐⭐

---

#### 8. **DMI（趋势强弱指标）**
**用途**：判断趋势是否明显
- ADX > 25 → 趋势明显
- ADX < 20 → 无明显趋势（盘整）
- +DI > -DI → 上升趋势
- -DI > +DI → 下降趋势

**实现难度**：⭐⭐⭐ 中等
**实用价值**：⭐⭐⭐⭐
